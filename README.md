# nanoFold Data Efficiency Competition

Data in biology is scarce and expensive/time consuming to generate, so we need to develop AI models that maximize per-sample efficiency. This competition is designed to encourage new innovation in data augmentation, model architecture, optimizer usage, and loss functions in order to maximize the learning efficiency of protein structure prediction models.

10k protein structures, no time limit, best val loss wins!

Inspired by [NanoGPT Slowrun](https://github.com/qlabs-eng/slowrun)
